# Todo App — Example Project

## Goal

Build a simple full-stack todo application with REST API and web frontend.

---

## Tech Stack

- Backend: Python 3.12 + FastAPI
- Frontend: React + TypeScript + Vite
- Database: SQLite (for simplicity)

---

## Acceptance Criteria

- [ ] API: CRUD endpoints for todos (/api/todos)
- [ ] Frontend: List, create, toggle, delete todos
- [ ] Tests pass with pytest
- [ ] Frontend builds without errors
