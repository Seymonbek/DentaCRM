# DentaCRM frontend

React 18 + TypeScript + Vite SPA. Task T1 provides only the container skeleton;
the full scaffold (routing, Tailwind, shadcn/ui, TanStack Query, Zustand,
axios, RHF+Zod, Recharts) is generated in task T5.

## Local development

```bash
docker compose up frontend
# → http://localhost:5173
```

Or on the host directly (after task T5 lands the real package.json):

```bash
npm install
npm run dev
```
